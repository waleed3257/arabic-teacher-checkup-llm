```
Leqa/
├── input/                           # 🆕 مجلد الإدخال
│   ├── id.xlsx                     # ملف أرقام الهوية
│   └── README.md                   # دليل الاستخدام
│
├── output/                          # 🆕 مجلد الإخراج
│   ├── [ID]_[timestamp].xlsx      # نتائج فردية
│   └── bulk_report_[timestamp].xlsx # نتائج دفعية
│
├── main.py                          # البرنامج الأساسي (فردي فقط)
├── main_enhanced.py                 # 🆕 البرنامج المحسن (فردي + دفعي)
│
└── services/
    └── excel_handler.py             # 🆕 معالجة ملفات Excel
```

---

## 🎨 تنسيق ملفات الإخراج

جميع ملفات Excel تحتوي على **3 أعمدة**:

### 1. ID
رقم الهوية الوطنية (10 أرقام)

### 2. Name  
الاسم الكامل للمعلم/المعلمة

### 3. Full Text
التقرير الوصفي الكامل المولد بواسطة الذكاء الاصطناعي

---

## 📊 مقارنة بين الطريقتين

| الميزة | فردي (Individual) | دفعي (Bulk) |
|--------|-------------------|-------------|
| **الإدخال** | يدوي من الطرفية | من ملف Excel |
| **عدد الأرقام** | 1 | متعدد (غير محدود) |
| **العرض في الطرفية** | ✅ كامل | ✅ ملخص |
| **ملف الإخراج** | ملف لكل رقم | ملف واحد لجميع الأرقام |
| **اسم الملف** | `[ID]_[timestamp].xlsx` | `bulk_report_[timestamp].xlsx` |
| **الوقت** | ~10-20 ثانية | ~10-20 ثانية × عدد الأرقام |

## 🔧 الملفات الجديدة

### 1. main_enhanced.py
البرنامج الرئيسي المحسن مع:
- قائمة تفاعلية
- دعم الفردي والدفعي
- حفظ تلقائي في Excel
- مؤشرات تقدم واضحة

### 2. services/excel_handler.py
وظائف معالجة Excel:
- `read_ids_from_excel()` - قراءة أرقام الهوية
- `save_individual_result()` - حفظ نتيجة فردية
- `save_bulk_results()` - حفظ نتائج دفعية
- `ensure_directories()` - إنشاء المجلدات

### 3. USAGE_GUIDE.md
دليل استخدام شامل مع:
- أمثلة تفصيلية
- استكشاف الأخطاء
- نصائح للأداء الأمثل

### 4. CHANGELOG.md
سجل التغييرات والإصدارات

### 5. input/README.md
دليل تنسيق ملف الإدخال

## 🎯 حالات الاستخدام

### استخدم الفردي (Individual) عندما:
- تريد معالجة رقم واحد فقط
- تريد رؤية التفاصيل الكاملة في الطرفية
- تريد التحقق من نتيجة معلم معين

### استخدم الدفعي (Bulk) عندما:
- لديك عدة أرقام (10+)
- تريد معالجة دفعة كاملة
- تريد ملف Excel واحد بجميع النتائج
- تريد توفير الوقت
