# برنامج توليد التقارير الوصفية للمعلمين
# Arabic Teacher Checkup Narrative Generator

برنامج ذكي لتوليد تقارير وصفية احترافية باللغة العربية عن المعلمين باستخدام الذكاء الاصطناعي المحلي (Ollama).

## المميزات

- 🔍 البحث عن المعلمين باستخدام رقم الهوية الوطنية
- 📊 دمج البيانات من 4 ملفات Excel مختلفة
- 🤖 توليد تقارير وصفية احترافية باللغة العربية باستخدام Ollama
- 📝 تفاصيل شاملة عن الحالة الوظيفية وبرامج النقل (فرص)
- ⚡ معالجة سريعة وواجهة سطر أوامر بسيطة

## متطلبات التشغيل

### 1. تثبيت Python
تأكد من تثبيت Python 3.11 أو أحدث:
```bash
python3 --version
```

### 2. تثبيت Ollama
قم بتحميل وتثبيت Ollama من: https://ollama.ai

```bash
# تشغيل خادم Ollama
ollama serve

# في نافذة طرفية أخرى، قم بتحميل النموذج
ollama pull gemma2:e4d
```

### 3. تثبيت المكتبات المطلوبة
```bash
cd /Users/waleedalhojaili/Dev/Leqa
pip install -r requirements.txt
```

## ملفات Excel المطلوبة

يجب أن تكون الملفات التالية موجودة في مجلد `data/`:

1. **teachers.xlsx** - البيانات الأساسية للمعلمين
   - السجل المدني، الاسم، الرتبة، التخصص
   - المدرسة، القطاع، الإدارة التعليمية
   - سنة المباشرة، نوع العقد

2. **Application.xlsx** - طلبات برنامج فرص
   - الإعلان (اسم البرنامج)
   - الرغبات (حتى 10 رغبات)
   - نقاط التطوير المهني والرخصة المهنية

3. **results.xlsx** - نتائج برنامج فرص
   - حالة الترشيح
   - إجمالي النقاط
   - أقل نقاط للنقل

4. **moved.xlsx** - بيانات المنقولين (اختياري)

## الإعدادات

يمكنك تعديل الإعدادات في ملف `config.py`:

```python
# تغيير النموذج اللغوي
OLLAMA_MODEL = "gemma2:e4d"  # أو أي نموذج آخر

# تغيير عنوان خادم Ollama
OLLAMA_URL = "http://localhost:11434/api/generate"

# تعديل النص التوجيهي للنموذج
SYSTEM_PROMPT = "..."
```

## استكشاف الأخطاء

### خطأ: تعذّر الاتصال بـ Ollama
```bash
# تأكد من تشغيل Ollama
ollama serve
```

### خطأ: النموذج غير موجود
```bash
# قم بتحميل النموذج
ollama pull gemma2:e4d

# تحقق من النماذج المحملة
ollama list
```

### خطأ: ملفات Excel غير موجودة
تأكد من وجود جميع ملفات Excel في مجلد `data/`

### خطأ: لم يتم العثور على المعلم
تحقق من:
- صحة رقم الهوية (10 أرقام)
- وجود الرقم في ملف teachers.xlsx
