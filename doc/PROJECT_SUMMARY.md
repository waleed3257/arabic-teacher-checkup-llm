## 🚀 كيفية الاستخدام

### الطريقة السريعة
```bash
# 1. تثبيت المتطلبات
pip install -r requirements.txt

# 2. تشغيل Ollama
ollama serve
ollama pull gemma2:e4d

# 3. اختبار النظام
python3 test_system.py

# 4. تشغيل البرنامج
python3 main.py
```

### الاستخدام البرمجي
```python
from services import get_teacher_by_id, build_prompt, generate_report

teacher = get_teacher_by_id("1065874370")
prompt = build_prompt(teacher)
report = generate_report(prompt)
print(report)
```
