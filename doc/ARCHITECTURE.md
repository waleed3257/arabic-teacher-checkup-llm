## تدفق البيانات | Data Flow

### 1. مرحلة الإدخال
```
المستخدم → رقم الهوية (10 أرقام) → Validator
```

### 2. مرحلة جمع البيانات
```
رقم الهوية → DataLoader
                  ↓
         ┌────────┴────────┐
         ↓                 ↓
    teachers.xlsx    Application.xlsx
         ↓                 ↓
         └────────┬────────┘
                  ↓
            results.xlsx
                  ↓
         Teacher Object (كامل)
```

### 3. مرحلة المعالجة
```
Teacher Object → PromptBuilder
                      ↓
              Structured Prompt
                      ↓
                 LLMClient
                      ↓
              Ollama API
```

### 4. مرحلة التوليد
```
Ollama (gemma2:e4d) → Arabic Narrative Report
                              ↓
                         عرض للمستخدم
```

## المكونات الرئيسية | Main Components

### 1. config.py
```python
# الإعدادات المركزية
- مسارات ملفات Excel
- عنوان Ollama API
- اسم النموذج اللغوي
- النص التوجيهي للنظام
```

### 2. models/teacher.py
```python
# نماذج البيانات
@dataclass Teacher:
    - البيانات الشخصية
    - البيانات الوظيفية
    - قائمة برامج فرص

@dataclass FursaProgram:
    - اسم البرنامج
    - الرغبات (حتى 10)
    - النقاط
    - حالة الترشيح
```

### 3. services/validators.py
```python
# التحقق من المدخلات
validate_national_id(id):
    - 10 أرقام
    - يبدأ بـ 1 أو 2
    - تنسيق صحيح
```

### 4. services/data_loader.py
```python
# تحميل ودمج البيانات
load_excel_files():
    - قراءة 4 ملفات Excel
    - تنظيف الأعمدة

get_teacher_by_id(id):
    - البحث في teachers.xlsx
    - جمع الطلبات من Application.xlsx
    - جمع النتائج من results.xlsx
    - دمج البيانات في كائن واحد
```

### 5. services/prompt_builder.py
```python
# بناء النص التوجيهي
build_prompt(teacher):
    - حساب سنوات الخدمة
    - تنسيق البيانات الأساسية
    - تفصيل البرنامج الأحدث
    - ملخص البرامج السابقة
    - إضافة التعليمات للـ LLM
```

### 6. services/llm_client.py
```python
# التواصل مع Ollama
generate_report(prompt):
    - إرسال POST request
    - معالجة الأخطاء
    - إرجاع النص المولد
```

### 7. main.py
```python
# التنسيق الرئيسي
main():
    1. طلب رقم الهوية
    2. التحقق من الصحة
    3. جلب البيانات
    4. بناء النص التوجيهي
    5. توليد التقرير
    6. عرض النتيجة
```
