# 🚀 دليل البدء | Getting Started

## مرحباً بك في نظام توليد التقارير الوصفية للمعلمين!

هذا دليل سريع لمساعدتك على البدء في استخدام النظام خلال دقائق.

---

## ⚡ البدء السريع (3 خطوات)

### الخطوة 1️⃣: تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### الخطوة 2️⃣: إعداد Ollama
```bash
# في نافذة طرفية منفصلة
ollama serve

# في نافذة أخرى
ollama pull gemma2:e4d
```

### الخطوة 3️⃣: تشغيل البرنامج
```bash
python3 main.py
```

**🎉 هذا كل شيء! أنت جاهز الآن.**

## 🔧 التخصيص

### تغيير النموذج اللغوي
عدل `config.py`:
```python
OLLAMA_MODEL = "llama3"  # بدلاً من gemma2:e4d
```

### تعديل أسلوب التقرير
عدل `SYSTEM_PROMPT` في `config.py`

---

## 💡 نصائح للاستخدام الأمثل

1. **تشغيل Ollama مسبقاً**: شغل `ollama serve` قبل استخدام البرنامج
2. **استخدم أرقام صحيحة**: تأكد من صحة رقم الهوية (10 أرقام)
3. **انتظر التوليد**: قد يستغرق توليد التقرير 5-15 ثانية
4. **اختبر أولاً**: استخدم `test_system.py` للتأكد من عمل النظام

---

## 🎓 الاستخدام البرمجي

يمكنك استخدام النظام برمجياً:

```python
from services import get_teacher_by_id, build_prompt, generate_report

# جلب بيانات المعلم
teacher = get_teacher_by_id("1065874370")

# بناء النص التوجيهي
prompt = build_prompt(teacher)

# توليد التقرير
report = generate_report(prompt)

print(report)
```
