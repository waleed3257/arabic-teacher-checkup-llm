# مجلد الإدخال | Input Folder

## ملف id.xlsx

هذا المجلد يحتوي على ملف `id.xlsx` الذي يُستخدم للمعالجة الدفعية (Bulk Processing).

### تنسيق الملف

يجب أن يحتوي الملف على عمود واحد على الأقل بأرقام الهوية:

| ID          |
|-------------|
| 1065874370  |
| 1040957142  |
| 1044846283  |
| ...         |

أو:

| السجل المدني |
|-------------|
| 1065874370  |
| 1040957142  |
| ...         |

### ملاحظات

- يمكن أن يكون اسم العمود: `ID`, `id`, `هوية`, `السجل المدني`, أو أي اسم آخر
- البرنامج سيبحث تلقائياً عن العمود الذي يحتوي على أرقام الهوية
- إذا لم يجد عمود مناسب، سيستخدم العمود الأول
- يجب أن تكون أرقام الهوية 10 أرقام وتبدأ بـ 1 أو 2

### الاستخدام

1. افتح ملف `id.xlsx`
2. أضف أرقام الهوية في العمود الأول
3. احفظ الملف
4. شغل البرنامج واختر الخيار 2 (معالجة دفعة)
