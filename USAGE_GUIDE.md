# دليل الاستخدام الشامل | Complete Usage Guide

## 🎯 طرق الاستخدام

يوفر البرنامج طريقتين للاستخدام:

### 1️⃣ البرنامج الأساسي (main.py)
معالجة فردية فقط - إدخال رقم هوية واحد في كل مرة

### 2️⃣ البرنامج المحسن (main_enhanced.py) ⭐ موصى به
يدعم:
- ✅ معالجة فردية (Individual)
- ✅ معالجة دفعة (Bulk) من ملف Excel

---

## 🚀 الاستخدام السريع

### البرنامج المحسن (موصى به)
```bash
python3 main_enhanced.py
```

ثم اختر:
- `1` للمعالجة الفردية
- `2` للمعالجة الدفعية
- `0` للخروج

---

## 📝 المعالجة الفردية (Individual)

### الخطوات:
1. شغل البرنامج: `python3 main_enhanced.py`
2. اختر `1` (معالجة فردية)
3. أدخل رقم الهوية (10 أرقام)
4. انتظر توليد التقرير

### المخرجات:
- ✅ **في الطرفية**: عرض كامل للبيانات والتقرير
- ✅ **ملف Excel**: يُحفظ في `output/[ID]_[timestamp].xlsx`

## 📦 المعالجة الدفعية (Bulk)

### الخطوات:

#### 1. تحضير ملف الإدخال
أنشئ أو عدل ملف `input/id.xlsx`:

| ID          |
|-------------|
| 1065874370  |
| 1040957142  |
| 1044846283  |

**ملاحظات:**
- يمكن أن يكون اسم العمود: `ID`, `id`, `هوية`, `السجل المدني`
- البرنامج سيكتشف العمود تلقائياً
- يجب أن تكون الأرقام 10 أرقام وتبدأ بـ 1 أو 2

#### 2. تشغيل المعالجة الدفعية
```bash
python3 main_enhanced.py
```
اختر `2` (معالجة دفعة)

#### 3. انتظر المعالجة
سيعالج البرنامج جميع الأرقام واحداً تلو الآخر:

## 🎨 تنسيق ملفات الإخراج

جميع ملفات الإخراج تحتوي على 3 أعمدة:

### 1. ID
رقم الهوية الوطنية (10 أرقام)

### 2. Name
الاسم الكامل للمعلم/المعلمة

### 3. Full Text
التقرير الوصفي الكامل المولد بواسطة الذكاء الاصطناعي

## 🔍 استكشاف الأخطاء

### خطأ: "ملف الإدخال غير موجود"
**الحل:**
```bash
# تأكد من وجود الملف
ls input/id.xlsx

# إذا لم يكن موجوداً، أنشئه
# افتح Excel وأنشئ ملف جديد في input/id.xlsx
```

### خطأ: "لا توجد أرقام هوية في الملف"
**الحل:**
- تأكد من وجود بيانات في العمود الأول
- تأكد من أن الأرقام ليست فارغة
- تأكد من حفظ الملف بعد التعديل

### خطأ: "رقم غير صالح"
**الحل:**
- تأكد من أن الرقم 10 أرقام
- تأكد من أن الرقم يبدأ بـ 1 أو 2
- تأكد من عدم وجود مسافات أو أحرف

### خطأ: "غير موجود"
**الحل:**
- تأكد من وجود الرقم في ملف `data/teachers.xlsx`
- تحقق من صحة رقم الهوية

## 📊 نصائح للأداء الأمثل

### للمعالجة الدفعية:
1. **قسم الدفعات الكبيرة**: إذا كان لديك أكثر من 100 رقم، قسمهم إلى ملفات أصغر
2. **تشغيل Ollama مسبقاً**: شغل `ollama serve` قبل البدء
3. **تجنب المقاطعة**: لا توقف البرنامج أثناء المعالجة
4. **راقب الذاكرة**: تأكد من وجود ذاكرة كافية

### للمعالجة الفردية:
1. **تحقق من الرقم**: تأكد من صحة الرقم قبل الإدخال
2. **انتظر الانتهاء**: لا تغلق البرنامج حتى ينتهي التوليد

## 🎓 الاستخدام البرمجي

يمكنك استخدام الوظائف مباشرة في كودك:

```python
from services import (
    read_ids_from_excel,
    get_teacher_by_id,
    build_prompt,
    generate_report,
    save_bulk_results
)

# قراءة الأرقام
ids = read_ids_from_excel('input/id.xlsx')

# معالجة كل رقم
results = []
for national_id in ids:
    teacher = get_teacher_by_id(national_id)
    if teacher:
        prompt = build_prompt(teacher)
        report = generate_report(prompt)
        results.append({
            'id': national_id,
            'name': teacher.full_name,
            'report': report
        })

# حفظ النتائج
save_bulk_results(results)
```
