# دليل البدء السريع | Quick Start Guide

## التثبيت والإعداد

### 1. تثبيت المتطلبات
```bash
# تثبيت مكتبات Python
pip install -r requirements.txt

# تثبيت Ollama (إذا لم يكن مثبتاً)
# قم بزيارة: https://ollama.ai
```

### 2. إعداد Ollama
```bash
# في نافذة طرفية منفصلة، شغل خادم Ollama
ollama serve

# في نافذة طرفية أخرى، حمل النموذج
ollama pull gemma2:e4d

# تحقق من النماذج المحملة
ollama list
```

### 3. اختبار النظام
```bash
# اختبر النظام بدون Ollama أولاً
python3 test_system.py

# إذا نجحت جميع الاختبارات، جرب البرنامج الرئيسي
python3 main.py
```

## الاستخدام

### الطريقة الأولى: التشغيل التفاعلي
```bash
python3 main.py
```
ثم أدخل رقم الهوية عندما يُطلب منك.

### الطريقة الثانية: استخدام Python مباشرة
```python
from services import get_teacher_by_id, build_prompt, generate_report

# جلب بيانات المعلم
teacher = get_teacher_by_id("1065874370")

# بناء النص التوجيهي
prompt = build_prompt(teacher)

# توليد التقرير
report = generate_report(prompt)

print(report)
```

## استكشاف الأخطاء الشائعة

### خطأ: "تعذّر الاتصال بـ Ollama"
**الحل:**
```bash
# تأكد من تشغيل Ollama في نافذة منفصلة
ollama serve
```

### خطأ: "النموذج غير موجود"
**الحل:**
```bash
# حمل النموذج
ollama pull gemma2:e4d

# أو استخدم نموذج آخر وعدل config.py
ollama pull llama3
```

### خطأ: "لم يتم العثور على ملفات Excel"
**الحل:**
تأكد من وجود الملفات التالية في مجلد `data/`:
- `teachers.xlsx`
- `Application.xlsx`
- `results.xlsx`
- `moved.xlsx`

### خطأ: "لم يتم العثور على المعلم"
**الحل:**
- تحقق من صحة رقم الهوية (10 أرقام)
- تأكد من وجود الرقم في ملف `teachers.xlsx`

## تخصيص النظام

### تغيير النموذج اللغوي
عدل ملف `config.py`:
```python
OLLAMA_MODEL = "llama3"  # أو أي نموذج آخر
```

### تعديل النص التوجيهي
عدل `SYSTEM_PROMPT` في ملف `config.py` لتغيير أسلوب التقرير.

### تغيير مسارات ملفات Excel
عدل `EXCEL_FILES` في ملف `config.py` إذا كانت الملفات في مكان آخر.

## الدعم الفني

للمساعدة أو الإبلاغ عن مشاكل، يرجى التواصل مع فريق التطوير.
